**Comment jouer à Maker's Pencil**

*Version offline*

 Pour ceux qui possède les fichiers complets sur leur ordinateur, il suffit d'ouvrir le fichier index.html avec firefox ou chrome.

*version en ligne*

 Pour jouer au jeu directement sur internet, il faut simplement se rendre à cette adresse sur un navigateur web (firefox ou chrome sont vivement conseillés) : cavalcadeur.github.io/zelda_maker
